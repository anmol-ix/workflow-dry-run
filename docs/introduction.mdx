## What is DelhiMart OS?

DelhiMart OS is a **Marketplace Operations System (CommerceOS)** for a hyperlocal multi-vendor marketplace with an **operator-first** mindset.

This MVP slice includes:

- **Backend API**: Express + TypeScript server with health endpoint
- **Frontend Web**: React + Vite application with health status UI
- **Monorepo Structure**: Managed with pnpm workspaces
